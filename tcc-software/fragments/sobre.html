O Software

Muitas Feiras ou Mostras não tem acesso a divulgação de modo geral, e necessitam de público, avaliadores e participantes para se manterem em atividade. A maioria das mostras não tem divulgação suficiente, e muito menos um sistema de gestão da mesma. Todos os processos nestas mostras são feitos manualmente como inscrição, avaliação até a divulgação que na maioria das vezes é feita oralmente.

Foi desenvolvido um software Open Source para a divulgação de projetos de mostras e para uso como ferramenta de gestão online. Também pode se consultar todas os projetos cadastrados, podendo filtrar por eixo, assim usuários encontram projetos próximos a sua escolha de interesse e visualiza seus dados detalhados com opção de entrar em contato com os autores.

O Software foi desenvolvido com uso de Node.js e MongoDB no back-end onde cada funcionalidade é exposta como um micro serviço REST. No front-end o framework JavaScript escolhido foi Angular.js utilizando de templates disponíveis através do também framework JavaScript, Bootstrap pois responsividade é disponibilizada out-of-the-box.
